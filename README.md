# To‑Do List API

A simple RESTful To‑Do List API built with Go.  
Users can register, log in, and manage their personal to‑dos with basic CRUD operations.

## Features

- User registration and login with JWT authentication  
- Create personal to‑dos  
- Get all to‑dos for the authenticated user  
- Update a to‑do  
- Delete a to‑do  
- Users can only access or modify their own to‑dos  

## Technologies Used

- Go (Golang)  
- JWT for authentication  
- bcrypt for password hashing  
- SQL or in‑memory database
